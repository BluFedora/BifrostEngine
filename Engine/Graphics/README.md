# BluFedora Graphics Abstraction

This is a thin abstraction over Vulkan and OpenGL to allow for easier management
cross-platform of graphics.

## Dependencies

- Core
- Data Structures
- [Memory][Memory]
- [Platform][Platform]
- TMPUtils
